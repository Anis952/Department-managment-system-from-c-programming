# This is the small project created from C programming

## About this project

##### This project is about the billing system of department store, which deals with the work of creating bills of the goods and items which are being purchased by the customer in the store. This project is useful in malls, shops, department store for keeping the record of the sales made by them and make the work of recording data`s reliable and easy.
